## Features

- [x] Nodejs server with Express
- [x] Unit tests with coverage
- [x] GitHub Actions + Codecov integration
- [x] Dockerfile that works on GCP CloudRun

## Installation

```sh
yarn  # Install dependencies.
yarn start  # Start server and watch for changes.
```
